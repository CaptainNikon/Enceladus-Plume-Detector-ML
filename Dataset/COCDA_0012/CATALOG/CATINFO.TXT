This directory contains the PDS catalog files for this data set.              
                                                                              
CATINFO.TXT  - The file you are reading.                                      
                                                                              
CDADS.CAT    - The catalog file for the dataset provided by the Cosmic        
               Dust Analyzer (CDA).                                           
                                                                              
CDAINST.CAT  - The catalog file for the instrument Cosmic Dust Analyzer       
               (CDA).                                                         
                                                                              
CDAREFS.CAT  - The catalog file for the references used on the data set       
               and its products.                                              
                                                                              
DUST.CAT     - The catalog file for the instrument target: DUST.              
                                                                              
INSTHOST.CAT - The catalog file for the instrument host: the CASSINI          
               spacecraft.                                                    
                                                                              
MISSION.CAT  - The catalog file for the CASSINI mission.                      
                                                                              
PROJREF.CAT  - The catalog file for the references used for the CASSINI       
               project.
